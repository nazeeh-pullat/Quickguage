QuickGauge is a lightweight, customizable hardware monitoring overlay for Windows. 

It displays real-time CPU, GPU, RAM, disk, temperature, and custom sensor data, with adjustable colors, layout, thresholds, size, opacity, hotkeys, tray controls, and startup options.

Built in C# and WPF, it is portable, dependency-free, and requires no installer.

### Features

* Live CPU, GPU, RAM, disk, and temperature monitoring
* Custom hardware sensors
* Color-coded status values
* Lightweight overlay
* System tray support
* Start with Windows
* No installer required

### Controls

* Show or hide each metric
* Reorder and rename rows
* Change colors and warning limits
* Adjust text size, scale, and opacity
* Drag the overlay anywhere
* Use a hotkey to show or hide it
* Double-click to open Settings
* Changes save automatically
